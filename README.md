# focus

focus is a time tracking app made with react & firebase. it's live at [boratto.ca/focus](https://boratto.ca/focus)

## Deployment

This app is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

### Manual Deployment

To manually deploy to GitHub Pages, run:

```bash
npm run deploy
```

This will build the app and push it to the `gh-pages` branch.

### GitHub Pages URL

Once deployed, the app will be available at: https://b44ken.github.io/focus
